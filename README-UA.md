# 🌟 LearnLingo — Платформа пошуку викладачів іноземних мов

**LearnLingo** — це сучасний, швидкий та інтерактивний веб-додаток для пошуку репетиторів іноземних мов та бронювання пробних занять. Проєкт побудований на базі **React 19**, **TypeScript** та **Vite** з повною інтеграцією сервісів **Firebase** (автентифікація, збереження даних у реальному часі та синхронізація списку обраного).

## ✨ Основні можливості проєкту

- **🏠 Головна сторінка**: Сучасний інтерфейс із яскравим дизайном, ключовими статистичними показниками платформи (кількість викладачів, відгуків, предметів) та швидким переходом до каталогу.
- **🔍 Гнучка фільтрація**: Можливість миттєвого пошуку викладачів за мовою викладання, рівнем володіння мовою (A1-C2) та погодинною оплатою.
- **⚡ Інтерактивні картки викладачів**: Відображення детальної інформації про викладача — мови, досвід, відгуки учнів, прайс, а також статус перебування в мережі (online/offline).
- **❤️ Список обраних (Favorites)**: Можливість додавати репетиторів до обраного. Дані синхронізуються з базою даних Firestore окремо для кожного авторизованого користувача.
- **📅 Бронювання пробного уроку**: Спливаюче модальне вікно з формою бронювання, вибором мети навчання (кар'єра, подорожі, хобі, тощо) та валідацією контактних даних.
- **🔐 Автентифікація**: Повний цикл реєстрації та авторизації користувачів через Firebase Authentication із миттєвими спливаючими сповіщеннями про статус входу.

## 🛠️ Технологічний стек

- **Core**: React 19, TypeScript, Vite
- **Стилізація**: Vanilla CSS + CSS Modules (для ізоляції стилів компонентів та забезпечення високої продуктивності)
- **Маршрутизація**: React Router DOM (використовується `HashRouter` для сумісності з GitHub Pages)
- **База даних та Сервер**:
  - **Firebase Authentication**: Для захищеної реєстрації та входу користувачів.
  - **Firebase Realtime Database**: Для зберігання та швидкого отримання загального списку викладачів.
  - **Firebase Cloud Firestore**: Для індивідуального збереження списку обраних викладачів кожного користувача.
- **Форми та Валідація**: React Hook Form, Resolvers, Yup (для безпечного введення даних та відображення помилок)
- **Сповіщення**: React Hot Toast (стильні та зручні спливаючі повідомлення)
- **Шрифти та Іконки**: `@fontsource/roboto`, `react-icons`

## 🚀 Встановлення та запуск локально

Щоб запустити проєкт на локальному комп'ютері, виконайте такі кроки:

### 1. Клонування репозиторію
```bash
git clone https://github.com/denis-ovcharov/language-teachers-react.git
cd language-teachers-react
```

### 2. Встановлення залежностей
```bash
npm install
```

### 3. Налаштування змінних оточення (Environment Variables)
Створіть файл `.env` у кореневій директорії проєкту за прикладом нижче та вкажіть ваші ключі Firebase:
```env
VITE_FIREBASE_API_KEY=ваша_api_key
VITE_FIREBASE_AUTH_DOMAIN=ваш_auth_domain
VITE_FIREBASE_DATABASE_URL=ваш_database_url
VITE_FIREBASE_PROJECT_ID=ваш_project_id
VITE_FIREBASE_STORAGE_BUCKET=ваш_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=ваш_messaging_sender_id
VITE_FIREBASE_APP_ID=ваш_app_id
```

### 4. Запуск у режимі розробки
Запустіть локальний сервер розробки:
```bash
npm run dev
```
Проєкт буде доступний за адресою: `http://localhost:5173/language-teachers-react/`

### 5. Збірка для продакшену
Для підготовки проєкту до деплою виконайте збірку:
```bash
npm run build
```
Після збірки ви можете перевірити результат локально за допомогою команди:
```bash
npm run preview
```

## 📂 Структура проєкту

```text
src/
├── components/          # Базові та перевикористовувані компоненти інтерфейсу
│   ├── App/             # Кореневий компонент додатку та роутинг
│   ├── Header/          # Хедер із навігацією та кнопками авторизації
│   ├── Modal/           # Модальне вікно-обгортка
│   ├── LoginForm/       # Форма входу
│   ├── RegistrationForm/# Форма реєстрації
│   ├── TeacherCard/     # Картка викладача
│   ├── TeachersList/    # Список викладачів з пагінацією
│   └── FiltersBox/      # Фільтри для сортування викладачів
├── context/             # Контексти (AuthContext для керування станом користувача)
├── lib/                 # Конфігурація сторонніх сервісів (firebase.ts)
├── pages/               # Сторінки додатку
│   ├── Home/            # Головна сторінка проєкту
│   ├── Teachers/        # Сторінка каталогу викладачів
│   └── Favorites/       # Сторінка обраних викладачів (приватний роут)
├── types/               # TypeScript описи типів (Teacher, User тощо)
├── global.css           # Глобальні CSS змінні та базові стилі
└── main.tsx             # Точка входу в додаток
```
